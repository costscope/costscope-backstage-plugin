# Costscope Backstage Plugin (package)

Note: This package expects the host Backstage app to provide Material UI dependencies.

Peer dependencies required by the plugin (install in the host app):

- @mui/material@^5.17.0
- @mui/icons-material@^5.17.0
- @mui/utils@^5.17.0

If your Backstage app doesn't already include these, add them to the host app's dependencies to ensure the plugin works correctly.
